Do Formula 1 Cars get faster over time?

How much faster do F1 tracks get over time?

With the help of the F1 Dataset provided at Kaggle, we are going to explore how much faster F1 cars have gotten over time.

The datasets we are going to use are races, which includes data about the specific races, like name of the circuit, year of the GP. laps provides lap times for each GP, each driver and each round in that GP. The last dataset we are going to use, results includes the results for each GP and driver. The strategy is to find different variables of interest from the differnt dataframes and bind them using the handy raceId, that is provided in each dataset.

Create a Dataset with races and mean lap times.
avg_lap <- laps %>%
  group_by(raceId) %>%
  filter(lap == round(mean(laps$lap, na.rm = T))) %>%
  summarise(mean_race = mean(sec_round, na.rm = T))  # Calculate the mean lap time per race for a mean lap


races$raceId <- as.character(races$raceId)
f1 <- left_join(races, avg_lap,by = c("raceId")) %>% select("raceId", "name", "mean_race", "year" )

f1$name2 <- str_replace_all(f1$name, "Grand Prix", "")

Next step: add the result dataset with the fastest lap time.

res <- read.csv("data/f1/results.csv")
res$raceId <- as.character(res$raceId)
res$fastest <- lubridate::ms(res$fastestLapTime)
## Warning in .parse_hms(..., order = "MS", quiet = quiet): Some strings
## failed to parse, or all strings are NAs
res$fastest <- as.duration(res$fastest)
res$fastest <- as.numeric(res$fastest)
res2 <- res %>%
  filter(!(is.na(fastestLapTime)) & fastest > 1) %>%
  group_by(raceId) %>%
  summarise(mean_fastest = mean(fastest),na.rm = T)
f1 <- left_join(f1,res2,by="raceId")

f1 <- f1 %>% filter(year > 2004)
